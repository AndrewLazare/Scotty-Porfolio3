    Enter a series of 5 numbers and determine the following things:
    a. Which number is the least
    b. Which number is the greatest
    c. The mean of the numbers
    d. The sum of all the numbers
    e. The product of all the numbers






                 $('#btnMain').click(function () {


            var num1 = Number($("#input1").val());
            var num2 = Number($("#input2").val());
            var num3 = Number($("#input3").val());
            var num4 = Number($("#input4").val());
            var num5 = Number($("#input5").val());

            //Step 2: Make Calculations with the data
            var sum = num1 + num2 + num3 + num4 + num5;
            var smallest = Math.min(num1, num2, num3, num4, num5);
            var largest = Math.max(num1, num2, num3, num4, num5);
            var product = num1 * num2 * num3 * num4 * num5;
            var average = sum / 5;

          


            //Step 3: Out the result
            var sumOutput = ("The sum of your numbers is " + sum);
            $("#resultsSum").text(sumOutput);

            var smallOutput = ("The smallest number is " + smallest);
            $("#resultsSmall").text(smallOutput);

            var largeOutput = ("The largest number is " + largest);
            $("#resultsLarge").text(largeOutput);

            var productOutput = ("The product of your numbers is " + product);
            $("#resultsProduct").text(productOutput);

            var averageOutput = ("The average of your numbers is " + average);
            $("#resultsAverage").text(averageOutput);
        });

    Fizz Buzz

    Enter two nubers and the program will iterate through all numbers from 1 to 100, printing numbers in ascending order to the screen. For each number that is a multiple of the first number input by the user, print "Fizz" instead of the number. For each number that is a multiple of the second number input by the user, print "Buzz" instead of the number. For numbers that are multiples of both numbers, print "FizzBuzz" instead of the number. 



               
               
            $("#btnFizz").click(function () {
                var returnValue = "";
                var value1 = Number($("#inputf1").val());
                var value2 = Number($("#inputf2").val());

                for (let i = 1; i <= 100; i++) {
                    if ((i % value1 == 0) && (i % value2 == 0)) {
                        returnValue += 'FizzBuzz ';
                    }
                    else if (i % value1 == 0) {
                        returnValue += 'Fizz ';
                    }
                    else if (i % value2 == 0) {
                        returnValue += 'Buzz ';
                    }
                    else {
      
                  returnValue += i + ' ';
                    }
                }

                $("#fbResults").text(returnValue);
            });

    Negative and Postitive

    Find the factorial of the given number. 


               
                $("#btnFact").click(function () {


                    var num = ($("#inputFact").val());
                    for (x = (num - 1); x > 1; x--) {
                        num = (num * x);
                    }
                    var outFact = ("The factorial of your number is: " + num);
                    $("#resultsFact").text(outFact);

                });

                $("#btnClearf").click(function () {
                    $("#inputFact").val("");
                    $("#resultsFact").text("");
                });

    Backwards and Forwards

    Enter a word to discover whether or not it is a palindrome! 


               
      $("#btnPal").click(function () {

                var fWord = String($("#inputPal").val());
                var word = fWord.toLowerCase().replace(/\s/g, '');
                for (var n = word.length - 1, revWord = ""; n >= 0; n--) { 
                    revWord += word.substr(n, 1);
                }

                revWord = revWord.toLowerCase().replace(/\s/g, '');

                if (revWord == word) {
                    var outPal = (fWord + " is a palindrome!");
                    $("#resultsPal").text(outPal);
                }


                else {

                    var outPal = (fWord + " is not a palindrome!");
                    $("#resultsPal").text(outPal);

                }


            });
